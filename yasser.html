Yasse-RL Halaby

Project Overview

This is a project I worked on with my friend Saumya for an Intro to RL class at Princeton. The idea was to have an agent learn how to play squash. We named our agent Yasse-RL Halaby after Yasser El Halaby '06, four time intercollegiate individuals champion. There is still much work to do before Yasse-RL can compete with Yasser.
